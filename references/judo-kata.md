# Judo Kata: Reference Sources

Checked on 2026-05-17.

## Primary reference

### Kodokan Global: Kata

Source: <https://kdkjd.org/kata/>

Why it matters:
- Kodokan is the historic and technical reference institution for judo founded by Kano Jigoro.
- This page is an official English source.
- It gives a concise overview of the main kata adopted at the Kodokan and links to further kata material.

The page lists these nine kata:

1. Nage-no-Kata
2. Katame-no-Kata
3. Kime-no-Kata
4. Ju-no-Kata
5. Kodokan Goshin-jutsu
6. Itsutsu-no-Kata
7. Koshiki-no-Kata
8. Seiryoku-Zenyo-Kokumin-Taiiku
9. Kodomo-no-Kata

Useful linked materials on the same page:
- `Kodokan KATA Textbook`
- `Kata Practice Guide`

## Recommendation

For judo kata references in this repository, prefer the Kodokan Global kata page as the default English primary source.
